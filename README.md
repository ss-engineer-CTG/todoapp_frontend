# 統合ToDo管理アプリケーション

階層型リスト表示とタイムライン表示を統合したプロジェクト・タスク管理アプリケーション

## 機能

- 階層型プロジェクト・タスク管理
- ガントチャート形式のタイムライン表示
- ダーク/ライトテーマ対応
- キーボードショートカット対応
- データエクスポート機能

## 技術スタック

- **フロントエンド**: React/TypeScript
- **スタイリング**: Tailwind CSS + shadcn/ui
- **状態管理**: React Context + useReducer
- **実行環境**: Electron対応予定

## インストール

```bash
npm install
npm run dev
